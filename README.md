# Pool Advantage: Facility Characteristics and Performance in NCAA Swimming
**Lily Cramer — Rice University, SMGT 490, May 2026**

This repository contains all materials for my capstone research project 
examining the effect of pool characteristics on swimmer performance in 
NCAA Division I women's swimming.

## Live Application
🏊 https://wpc01l-lily-cramer.shinyapps.io/ncaa_pool_app/

## Repository Structure
- `paper/` — LaTeX source and figures for the final paper
- `code/` — Reproducible R scripts for data cleaning, EDA, and modeling
- `app/` — Shiny web application source code

## How to Run
1. Run scripts in `code/` in order (01 → 02 → 03)
2. Run `app/app.R` in RStudio or with `shiny::runApp("app/")`

## Data
Raw performance data from USA Swimming (2021–2026). 
Facility data collected from institutional websites.
